## Pourquoi je veux installer une IA locale ?

Une utilisation intelligente de l'IA peut être bénéfique pour apprendre plus rapidement.

Dans mon cas, l'IA locale sera utile pour :

- Corriger, rédiger, comprendre des lignes de Code
- Comprendre l'utilisation et fonctionnement des logiciels, commandes
- Réexpliquer des notions non comprises et faire des tests / exercices
- Avoir un point de vu extérieur qui permet de connaître de meilleurs options
- Générer des idées : que ce soit pour trouver un plan pour un rapport, écrire du texte etc.
- Rassembler les connaissances : utilisant GitHub pour marquer ce que j'apprend ou Obsidian, LM peut avoir accès à mes notes, créer des connections entre elles et les utiliser

Avoir une IA sur le PC permet d'avoir un deuxème cerveau local qui peut être utilisé hors-ligne.

C'est un assistant personnel polyvalent :

- Aucune censure : les modèles téléchargés n'ont pas les blocages stricts des IA commerciales.
- Confidentialité totale : envoyer des documents personnels, cours, données privées peuvent êtres partagés sans risquer que ces infos ne sortent du PC.

Les recherches internets restent importantes, il ne faut pas les oublier, mais l'IA est un OUTILS fort, qui permet de résumer rapidement des informations, surtout celles que nous avons nous même apportés, et d'être plus efficace sur certaines tâches.
