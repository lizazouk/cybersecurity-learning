# Créer un cadre avec logo comme ceci :

![Linux](https://img.shields.io/badge/Linux-FCC624?style=for-the-badge&logo=linux&logoColor=black)

Logos officiels et couleurs : https://simpleicons.org/?q=notion

**La structure du lien**

La base du lien : `https://img.shields.io/badge/NOM-COULEUR?style=for-the-badge&logo=LOGO&logoColor=white`

Chercher les infos et remplacer :
- NOM : message affiché `Linux`
- COULEUR : `FCC624` penser à retirer le #
- LOGO : nom du logo inscrit `linux`
- COULEUR DU LOGO : si COULEUR est foncé alors `=white` , sinon `=black`

Une fois le lien créer, rajouter `![]()`

Et Voilà !
